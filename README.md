# Click-Go---Programacion
Este repositorio almacena todos los códigos del proyecto de Bases de Datos de **Martín Zonis**, **Kenai Jeiman** y **Thiago Ducau**.
El proyecto consiste en una botonera la cual busca ayudar a incluir a los adultos de la tercera edad asi como personas con otras dificultades.
Este repositorio cuenta con 5 archivos los cuales deben ser ejecutados en los siguientes directorios: "*logo.png*" y "*interfaz.py*" deben estar en la misma carpeta, asi como "*arduino.ino*" debe estar en una carpeta con su mismo nombre.
